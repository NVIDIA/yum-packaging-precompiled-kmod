# yum packaging precompiled kmod

## DISTROS

The plan going forward is for Linux distro vendors to provide their own signed kernel modules for NVIDIA Open GPU flavor, e.g.

- Alma: `nvidia-open-kmod-580.105.08-2.el10_1.x86_64.rpm`
- Azure: `cuda-open-hwe-580.105.08-1_6.12.57.1.1.azl3.x86_64.rpm`
- CIQ: `kmod-nvidia-dc-open-latest-580.105.08-1.el10_1_ciq.x86_64.rpm`
- RHEL: `kmod-nvidia-open-580.105.08-6.12.0-124.20.1-580.105.08-3.el10_1.x86_64.rpm`
- SUSE: `nvidia-open-driver-G06-signed-cuda-kmp-default-580.105.08_k6.18.5_1-1.4.x86_64.rpm`
- Ubuntu: `linux-modules-nvidia-580-6.11.0-1016-nvidia_6.11.0-1016.16+2_amd64.deb`
